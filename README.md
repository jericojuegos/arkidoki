# Arkidoki

A React application to brainstorm and scaffold WordPress plugins.

## Features
- **Project Configuration**: Set project name, description, author, etc.
- **Module Management**: Select modules like Logger, Sync, Connections.
- **React Options**: Toggle Pagination, Filters, Modals.
- **Live Code Preview**: See the generated boilerplate code instantly with syntax highlighting.
- **Copy to Clipboard**: Easily copy the code to your files.

## Tech Stack
- React + TypeScript
- Vite
- Sass
- PrismJS (Syntax Highlighting)

## Usage

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```

## Structure
- `src/generator`: Core logic for code generation and templates.
- `src/components`: UI components (InputSection, PreviewSection).
- `src/styles`: SCSS styles.
